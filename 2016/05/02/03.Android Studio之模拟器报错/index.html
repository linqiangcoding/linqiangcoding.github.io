Error Unable to start the Genymotion virtual device.解决方法

Genymotion安卓模拟器虽然给我们测试软件或者做微信营销、网赚项目等带来了很大的便利，但是又经常出问题让我们煞费苦心，今天电脑知识网小编又将给大家分享如何解决：
Error Unable to start the Genymotion virtual device.这样的错误提示。

The Genymotion virtual device could not obtain an IP address.
For an unknown reason.VirtualBox DHCP has not assingned an IP address
to virtual device.Run the VirtualBox software to check for issues.
For more help refer to:https://cloud.genymotion.com/page/faq/#collapse-nostart
这段英文的大体意思是Genymotion安卓模拟器无法获得IP来运行虚拟机设备系统，让我们打开：VirtualBox虚拟机来检查设置。其实 Genymotion安卓模拟器运行是有宿主的，它的宿主就是：VirtualBox虚拟机，如果VirtualBox虚拟机设置有问 题，Genymotion就无法运行。
Error Unable to start the Genymotion virtual device.解决方法：
打开–Oracle VM VirtualBox–管理–全局设定–网络–点击下图所示的“+”号，增加一个新的虚拟网卡–再点击“+”号下面的毛笔图标编辑虚拟网卡–设置如下：


设置好后在Oracle VM VirtualBox选择您的安卓系统–设置–网络–选择你新增加的虚拟网卡–确定即可。
